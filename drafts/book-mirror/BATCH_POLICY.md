# Book Mirror Batch Policy

Source: Ramy, Discord Book Mirror thread, 2026-04-30.

- Process at most **5 books/articles per day** from the Readwise/Notion library.
- Do not try to mirror the full library in a single day.
- Prefer quality and source coverage over throughput.
- Keep a manifest of completed, skipped, sparse, and deferred items so the next daily batch can resume cleanly.
- If only highlights/summaries are available, label the output as partial/sparse rather than presenting it as a full chapter-by-chapter book mirror.
